# Project Memory

## Design Language Preferences
User wants TWO design references applied to future webapp projects:
1. **Bigfolio.co style** (baseline) — see [design-language.md](design-language.md)
2. **CSS Design Awards winners** (premium/advanced tier) — see [award-winning-patterns.md](award-winning-patterns.md)

3. **21st.dev component library** (React components) — see [21st-dev-components.md](21st-dev-components.md)
4. **Google Stitch MCP** (AI-generated Tailwind UI) — see [stitch-design-patterns.md](stitch-design-patterns.md)

Use Bigfolio as the default. Layer in award-winning patterns when building premium/immersive experiences or when user requests higher-end design. Use 21st.dev components as reference for React/Next.js projects (Radix UI + Tailwind + Framer Motion patterns). Use Stitch patterns for rapid Tailwind CSS prototyping, Material icons, glass morphism, and gradient effects.
